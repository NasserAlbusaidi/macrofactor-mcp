# Public positioning

MacroFactor MCP is a local **Fueling & Recovery Decision Engine**. It turns
nutrition and optional training data into four flagship decisions:

1. `recovery_check` — Am I recovered enough for today's training?
2. `daily_briefing` — What do today's fueling and recovery signals say together?
3. `weekly_report` — What should change next week?
4. `nutrition_performance_correlation` — Did my fueling support recovery and performance?

These four workflows are the complete v0.1 flagship promise. The server
registers 29 MCP tools in total; the other 25 support setup, import, inspection,
and deeper follow-up questions.
